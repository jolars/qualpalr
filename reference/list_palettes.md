# List available color palettes

List available color palettes

## Usage

``` r
list_palettes()
```

## Value

A list of available color palettes. Each palette is a named list with a
character vector.

## See also

[`qualpal()`](https://jolars.github.io/qualpalr/reference/qualpal.md)
